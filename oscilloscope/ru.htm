Политика конфиденциальности

Это приложение представляет собой мощный ручной осциллограф, простое в использовании, точное измерение.

Это приложение будет использовать ваш микрофон для записи, а все записанные данные будут использоваться только для отображения осциллографа и никогда не будут использоваться для других целей. Это приложение не подключается к сети, не требует никакой идентификационной информации пользователя и никогда не раскрывает конфиденциальность пользователя.
