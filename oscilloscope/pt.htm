Política de Privacidade

Esta aplicação é uma poderosa aplicação portátil de osciloscópio, fácil de usar e precisa.

Este aplicativo usará seu microfone para gravar e todos os dados gravados serão usados apenas para exibição no osciloscópio e nunca serão usados para outras finalidades. Este aplicativo não se conecta à rede, não exige nenhuma informação de identidade do usuário e nunca revelará a privacidade do usuário.
