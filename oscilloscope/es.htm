Política de privacidad

Esta aplicación es una poderosa aplicación de osciloscopio de mano, fácil de usar y de medición precisa.

Esta aplicación utilizará su micrófono para grabar, y todos los datos grabados solo se utilizarán para la visualización del osciloscopio y nunca se utilizarán para otros fines. Esta aplicación no se conecta a la red, no requiere ninguna información de identidad del usuario y nunca revelará la privacidad del usuario.
